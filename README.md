# 人物气质转译写作器

一个用于中文长文、公众号文章、产品测评和评论文章的 Codex Skill。

它的目标不是模仿名人，也不是冒充某个人写作，而是把公开可观察的表达气质转译成高层次、可复用、相对安全的写作特征，再用用户自己的账号身份生成原创文章。

## 它适合做什么

- 公众号长文
- 中文产品测评
- 产品、公司、事件评论
- AI 产品分析
- 访谈式批评文章
- 观点随笔
- 基于公开素材的长文写作

典型用法：

```text
用马斯克那种第一性原理和工程判断的气质，写一篇关于某个 AI 产品的评论。
不要冒充本人，不要写成商业稿，要尖锐一点。
```

```text
用易立竞式冷静追问的气质，写一篇公众号长文。
主题是：AI 陪伴产品到底是在解决孤独，还是在制造新的依赖。
```

## 它不做什么

- 不直接模仿在世人物的独特个人风格
- 不冒充指定人物本人
- 不伪造背书、授权、采访或引用
- 不编造产品体验、截图、测试结果或内部消息
- 不使用私人信息、八卦、人肉资料或未证实传闻推断风格
- 不输出高风险医疗、法律、金融或投资建议

## 核心思路

这个 Skill 会把用户指定的人物转译成高层表达特征，例如：

| 人物方向 | 可转译特征 |
| --- | --- |
| 马斯克 | 第一性原理、工程判断、系统瓶颈、资源约束、长期赌注 |
| 易立竞 | 冷静追问、短问题推进、拒绝讨好、边界感、让矛盾自己暴露 |
| 罗永浩 | 产品经理视角、消费者立场、自嘲幽默、强判断、口语化拆解 |
| 吴军 | 技术史脉络、系统因果链、耐心解释、克制但明确的判断 |

这些是气质转译，不是一比一仿写模板。

## 推荐输入格式

```text
用[人物名]的表达气质转译，不要冒充本人，写一篇[文章类型]。

主题：[主题]
素材：[公开资料、链接、产品介绍、采访片段、你的观点]
账号片尾：[你的账号片尾]
要求：[尖锐/克制/幽默/深度/少标题/多故事]
```

如果没有账号片尾，Skill 会使用占位符：

```text
> / 作者：[你的账号名]
> / 联系或投稿：[你的联系方式]
```

## 质量检查

Skill 内置四层质检：

- L1 硬规则，检查冒充、伪造背书、编造引用、盗用片尾等问题
- L2 风格一致性，检查气质转译是否稳定，是否滑向普通 AI 腔
- L3 内容质量，检查事实支撑、反方观点、限制条件和分析深度
- L4 人味，检查文章是否像写给真实读者，而不是提示词演示

同时还包含公众号合规风险检查：

- 侵权风险
- 虚假宣传
- 隐私泄露
- 诱导关注
- 敏感表达

## 安装方式

把 `SKILL.md` 放入你的 Codex Skills 目录，或作为项目内 Skill 文件使用。

一个常见目录结构：

```text
persona-style-writer/
├── SKILL.md
├── README.md
├── LICENSE
└── examples/
    └── persona-card-template.md
```

## 扩展人物气质库

不建议收集大量原文做仿写训练。更稳妥的方式是从公开材料中整理“人物气质卡”：

- 公开资料来源
- 高层表达特征
- 常见文章结构
- 提问和论证方式
- 情绪温度
- 禁止模仿项

模板见 [examples/persona-card-template.md](examples/persona-card-template.md)。

本项目用于原创写作辅助和公开表达气质研究，不应用于冒充他人、复制在世创作者的独特表达、伪造授权、批量复刻文章或规避平台规则。

使用公开材料时，请尊重原作者版权和平台规则。建议保存链接、摘要和风格观察，不要建立未经授权的全文语料库。
